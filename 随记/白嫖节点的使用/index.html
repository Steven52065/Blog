白嫖节点的使用

脑子一热就买了个服务器，为了不浪费，搭了个梯子
2022-12-23 22:15 schedule 3 min 50 s.

因为是 Vless 的节点，所以本文只会介绍 V2ray 相关软件的使用方法。其他客户端( Clash、Openwrt 等)就不写了

  • 节点仅保证最低限度的访问外网，即使用 Google、Youtube、Github 等，无法保证可以使用 New Bing、ChatGPT、Netflix 等对节点要求较高的网站。
  • 晚八点前后的网络高峰期速度慢是正常现象，可以通过使用优选 IP 来缓解

使用教程

Windows

  1. 下载 V2rayN 客户端 ，选择最新版本，下载 v2rayN.zip 文件，并解压
  2. 下载 Xray-core ,选择对应的系统、架构下载（如果不知道，那就是 Xray-windows-64.zip ），将解压出的文件放到 V2rayN 客户端 的文件目录下（即和 v2rayN.exe 文件在一起）。
  3. 启动 V2rayN ，复制节点信息后在 V2rayN 中右键，并选择“从剪贴板批量导入URL”，若出现“成功”二字，导入成功
  4. 右键节点，选择“测试服务器真连接延迟”若测试结果中出现延时，说明节点正常（用不了是你的问题）
  5. 开始使用吧

Android

  1. 下载 V2rayNG 客户端, github 或 Google play 选一。
  2. 复制节点信息后点击 V2rayNG 上方“+”，“从剪贴板导入”
  3. 点击右上角三个点，“测试全部配置真连接”，若出现测试结果，说明节点没问题（用不了是你的问题）
  4. 点击右下角的圆形图标，开始使用吧

Ios

  1. 搞个非国区帐号，并下载 Shadowrocket （国区里的是假的）
  2. 复制节点信息后打开 Shadowrocket 即可自动导入
  3. 打开上方开关，开始使用吧

有测试结果但节点无法使用

Windows

  • 点击“重启服务”试试
  • 关闭除 V2rayN 以外的代理软件和浏览器的代理插件并重启 V2rayN
  • 重启电脑
  • 换手机

Android

  • 重启手机
  • 换苹果

Ios

  • 重启手机
  • 换安卓

节点可用但速度慢

寻找优选 IP

1.自己找

因为使用了 Cloudflare CDN ，默认分配的节点可能比较慢（通常比较慢），可以在 Windows 下下载 CloudflareSpeedTest ,关闭所有代理软件后解压并双击运行 CloudflareST.exe 文件，等待测速完成。测速结果第一行就是既下载速度最快、又平均延迟最低的最快 IP

2.用现成的

在 在线成品优选ip网站 中根据你的运营商选择一个速度较快且延迟较低的 IP
注：延迟低不代表速度一定快，通常速度快比延迟低更重要

优选 IP 的使用

这个 IP 的测试结果仅在当前网络环境下有效，换了不一定好。优选 IP 可能会失效，若使用优选 IP 后无法连接节点，请去掉优选 IP 尝试或更换优选 IP

双击编辑节点，用优选到的 IP 替换到“地址”栏内的内容
