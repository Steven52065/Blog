WP BUUCTF 刷新过的图片
2023-02-02 19:58 schedule 34 s.

题目链接: https://buuoj.cn/challenges#%E5%88%B7%E6%96%B0%E8%BF%87%E7%9A%84%E5%9B%BE%E7%89%87

题目

浏览图片的时候刷新键有没有用呢

思路

提到了刷新的图片，可以想到 F5 隐写
使用工具 F5-steganography
使用命令

java Extract /图片的绝对路径 [-p 密码] [-e 输出文件]

可以解出一个伪加密的压缩包
打开压缩包后即可得到 flag
flag{96efd0a2037d06f34199e921079778ee}

为什么死活运行不了呢… Java 版本太新的问题？？？怪
